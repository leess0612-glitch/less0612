## 2. 엑셀 입력 규칙

### 컬럼 매핑 (확정)

| 출처 | 구글시트 열 | → 엑셀 열 | 값 |
|------|-----------|----------|----|
| 유선개통 | G (고객명) | D | 가운데 글자 * 처리 |
| 유선개통 | C (통신사) | E | 가공값 |
| 유선개통 | L (상품구분) | F | 가공값 |
| 유선개통 | H (은행) | G | |
| 렌탈개통 | G (고객명) | D | 가운데 글자 * 처리 |
| 렌탈개통 | — | E | **공란** |
| 렌탈개통 | (고정값) | F | **가전렌탈** |
| 렌탈개통 | H (은행) | G | |

### 고객명 정제 규칙 (마스킹 전 적용)
1. `/` 가 포함된 경우 `/` 이후 내용 전체 삭제
2. 숫자 전체 삭제
3. 앞뒤 공백 제거

예: `홍길동2/메모` → `홍길동` → 마스킹 적용

### 고객명 마스킹 규칙

| 글자수 | 예시 | 결과 |
|--------|------|------|
| 2글자 | 강민 | 강# |
| 3글자 | 홍길동 | 홍#동 |
| 4글자 이상 | 남궁민수 | 남##수 |

규칙: 첫 글자 + `#` × (글자수-2) + 마지막 글자 (2글자는 첫 글자 + `#`)

### 데이터 배치 순서 (3행부터)
1. 백업 데이터: A열이 비어있고 D열에 값 있는 행 → 위에서부터 최대 30개
2. 유선개통 신규 데이터
3. 렌탈개통 신규 데이터

### 정렬
- 기준: D열(고객명)
- 방향: 가나다 오름차순 (ㄱ→ㅎ)
- 범위: 3행부터 마지막 데이터 행까지 (3행 포함, 2행은 고정이므로 제외)

## 3. 이미지 캡처 (2단계 — 구현 완료)
- 고정 포함 행: 2행 (헤더)
- 캡처 범위: D2:G(마지막 데이터 행) — 선택된 백업 + 유선개통 + 렌탈개통 전체
- 저장 위치: 엑셀 파일과 같은 폴더
- 파일명: `YYYYMMDD_사은품명단.png`
- 방식: win32com CopyPicture → 클립보드 → PIL ImageGrab → PNG 저장

### 캡처 후 처리
- 신규 데이터를 무작위 순서로 재정렬
- 재정렬된 데이터를 시트 맨 아래에 복사 (A열 번호 없이)
- 복사 완료 후 원본 위치의 신규 데이터 삭제
