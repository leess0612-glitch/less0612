## 1. 최초 요청

특정 구글시트의 두 탭에 정리된 데이터를 추출해서
특정 엑셀 파일 시트에 설정된 규칙에 따라 입력한 후,
특정 행(항상 고정) + 새로 입력된 부분 범위를 스크린샷(이미지화)하고,
네이버카페 특정 게시판에 정의된 규칙대로 제목을 생성하여
특정 시간에 글쓰기를 하는 자동화 프로그램을 기획.

심각한 오류나 시스템 구현이 의도한 방향과 다를 것으로 예상되면 반자동화도 가능.
단, 반자동화로 구현할 경우 카카오톡으로 특정 시간마다 알림을 보내
버튼을 누르는 작업을 할 수 있도록 안내해야 함.

## 2. 세부 기획 Q&A

### 0. 엑셀 파일 위치
- `C:\Users\a\Desktop\안티그라비티\입금명단자동화\신청현황_자동화테스트버전.xlsx`

### 1. 구글시트
- 권한 있는 사람만 볼 수 있는 시트
- URL: `https://docs.google.com/spreadsheets/d/1y5wfMhcM3_S7FnJWHAhTIkqyzFZZCcItGQpVL5nu5XY`
- `렌탈개통`, `유선개통` 두 탭에서 데이터 추출

**유선개통 추출 규칙**
- C(통신사), G(고객명), H(은행), L(상품구분) 가져오기
- C열 가공: KT=KT, SKT=SK, SKB=SK, SK알뜰=SK, LGU+=LG, LG소호=LG, 스카이=스카이, LG헬로=LG헬로
- L열 가공: 인단=인터넷, 번들=인터넷+TV
- L열에 "유심" 포함된 행은 스킵

**렌탈개통 추출 규칙**
- G(고객명), H(은행) 가져오기

### 2. 엑셀 입력 규칙
- 신청현황.xlsx에 입력, D: 고객명 / E: 통신사 / F: 상품명 / G: 입금계좌 — 3행부터
- 가져온 행과 같은 행에 입력

**공통 규칙**
- A열에 숫자 없는 행 = 이전 백업 데이터. 위에서부터 순서대로 15~30개(무작위 개수)를
  3행부터 채운 뒤, 그 다음 행부터 유선개통 → 렌탈개통 순으로 배열
- G열 값 → D열, 가운데 글자는 마스킹 처리
  - 2글자: 강민 → 강*
  - 4글자 이상: 남궁민수 → 남**수
- D열(고객명) 기준 가나다 오름차순 정렬
- 스크린샷(이미지화) 완료 후, 입력된 값을 무작위 순서로 재정렬 → 맨 아래로 복사
  → 복사 완료되면 기존 입력값 전체 삭제

**개별 규칙**
- 렌탈개통: F열에 "가전렌탈" 고정 입력, E열은 공란
  (※최초 답변에서 E열로 잘못 기재했던 것을 F열로 정정)
- 유선개통: C열 값 → E열, L열 가공값 → F열, H열 값 → G열

### 3. 고정행
- 2행 (항상 캡처 범위에 포함)

### 4. 네이버카페 제목 규칙
- 형식: `YYYY-MM-DD 사은품지급 명단` (예: 2026-06-08 사은품지급 명단)

### 5. 글쓰기 시간
- 매일 특정 시간, 토·일요일 제외, 공휴일 제외
- 공휴일은 매년 자동 판단 필요

### 6. 카카오톡 알림
- 개인채팅 (나에게 보내기)
