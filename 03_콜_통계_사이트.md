# 모듈 3. 콜 통계 사이트

## 개요
고객 문의 인바운드 콜 데이터를 엑셀/CSV로 업로드하여
팀 전체의 콜 응대 현황을 시각화하고 분석하는 대시보드.

## 핵심 문제 정의
- 콜 기록이 엑셀에만 있어 트렌드 파악이 어려움
- 팀원별 응대량, 처리율 비교가 수동 집계로 번거로움
- 시간대별 피크 타임 파악 불가로 인력 배치 비효율

---

## 주요 기능

### 3-1. 데이터 업로드
- 엑셀(.xlsx, .xls) / CSV 파일 업로드
- 컬럼 매핑 UI: 파일의 어떤 컬럼이 "날짜", "담당자", "통화시간" 인지 사용자가 지정
- 업로드 이력 관리 (여러 파일 누적 가능)
- 중복 데이터 감지 및 경고

### 3-2. 지원 데이터 항목
| 항목 | 설명 |
|---|---|
| 날짜/시간 | 콜 발생 일시 |
| 담당자 | 응대한 팀원 이름 |
| 통화시간 | 초 또는 분 단위 |
| 처리결과 | 해결/미해결/콜백/부재 등 |
| 문의유형 | 배송/반품/상품문의 등 (선택) |
| 고객연락처 | 선택 (통계용, 개인정보 마스킹) |

### 3-3. 메인 대시보드

**요약 카드**
- 오늘 총 콜 수
- 평균 통화시간
- 해결률 (%)
- 미처리 건수

**차트**
- 일별 콜 수 추이 (라인 차트)
- 시간대별 콜 분포 히트맵 (0~24시)
- 담당자별 응대 건수 (바 차트)
- 처리결과 비율 (도넛 차트)
- 문의유형 분포 (파이 차트)

### 3-4. 담당자별 상세
- 담당자 선택 시 개인 통계 드릴다운
- 처리 건수, 평균 통화시간, 해결률
- 월별 추이

### 3-5. 기간 필터
- 오늘 / 이번 주 / 이번 달 / 지난 달
- 사용자 정의 날짜 범위 선택

### 3-6. 리포트 내보내기
- 현재 화면 PDF 저장
- 요약 데이터 엑셀 내보내기
- 차트 이미지 저장 (PNG)

---

## 화면 구성
```
[상단] 기간 필터 | 담당자 필터 | 데이터 업로드 버튼
[상단] 요약 카드 4개 (총 콜/평균시간/해결률/미처리)
[중간] 일별 추이 차트 (와이드)
[하단 좌] 시간대별 히트맵
[하단 중] 담당자별 바차트
[하단 우] 처리결과 도넛차트
```
