# 모듈 2. 이미지요소 합성기

## 개요
블로그 업로드용 상품 이미지를 빠르게 제작할 수 있는 캔버스 기반 편집 도구.
포토샵 없이 브라우저에서 바로 합성, 저장.

## 핵심 문제 정의
- 블로그마다 이미지 규격/스타일이 다름
- 매번 포토샵 열기 번거로움
- 같은 템플릿을 여러 상품에 반복 적용하는 작업이 많음

---

## 주요 기능

### 2-1. 캔버스 기본 설정
- 사이즈 프리셋: 블로그 표준(860×N), 정사각형(1:1), 세로형(3:4) 등
- 사용자 정의 사이즈 입력
- 배경색 / 배경 이미지 설정

### 2-2. 요소 추가
| 요소 | 세부 기능 |
|---|---|
| 이미지 | 로컬 파일 업로드, 투명도 조절, 크롭, 반전 |
| 텍스트 | 폰트 선택, 크기, 색상, 굵기, 그림자, 외곽선 |
| 도형 | 사각형, 원, 삼각형, 색상/테두리 설정 |
| 스티커 | 기본 제공 아이콘 세트 (화살표, 별, 뱃지 등) |
| 워터마크 | 반투명 텍스트/로고 삽입 |

### 2-3. 레이어 관리
- 레이어 목록 패널 (포토샵과 유사)
- 레이어 순서 드래그로 변경
- 레이어 잠금 / 숨김 / 삭제
- 레이어 복사

### 2-4. 편집 도구
- 드래그로 위치 이동
- 모서리 핸들로 크기 조절
- 회전 핸들
- 정렬: 좌/우/상/하/가운데 정렬, 균등 분배
- 실행 취소 / 재실행 (Ctrl+Z / Ctrl+Y)

### 2-5. 템플릿 시스템
- 현재 작업물을 템플릿으로 저장
- 템플릿 불러와서 이미지만 교체
- 기본 제공 템플릿: 상품 소개형, 가격 강조형, 비교형

### 2-6. 배치 처리
- 이미지 여러 장 업로드
- 같은 템플릿(텍스트, 워터마크 등)을 모든 이미지에 자동 적용
- 한번에 전체 저장

### 2-7. 내보내기
- JPG (품질 조절 슬라이더)
- PNG (투명 배경 지원)
- 블로그 최적화 사이즈 자동 압축 옵션

---

## 화면 구성
```
[상단] 파일 | 편집 | 뷰 | 내보내기
[좌측] 요소 추가 패널 (이미지/텍스트/도형/스티커)
[중앙] 캔버스 편집 영역
[우측] 레이어 패널 + 선택 요소 속성 패널
```
