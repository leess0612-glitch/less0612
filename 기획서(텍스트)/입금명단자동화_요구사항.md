# 입금명단 자동화 시스템 — 요구사항 정의서

## 시스템 개요
구글시트 데이터 → 엑셀 입력 → 범위 이미지화 → 네이버카페 자동 게시

---

## 확인 필요 항목

### 1. 구글시트 (데이터 소스)
- [ ] 시트 URL 또는 시트 ID
- [ ] 탭 1 이름:
- [ ] 탭 2 이름:
- [ ] 가져올 컬럼 목록 (예: A열=이름, B열=금액, C열=날짜 등)
- [ ] 데이터 시작 행 (헤더 제외 몇 번째 행부터?)
- [ ] 두 탭 데이터를 합치는 방식 (단순 병합? 조건 병합? 탭별로 별도 처리?)

---

### 2. 엑셀 파일 (출력 대상)
- [ ] 엑셀 파일 경로 및 파일명
- [ ] 데이터를 입력할 시트 이름
- [ ] 입력 시작 셀 (예: B3부터 시작)
- [ ] 컬럼별 입력 규칙 (구글시트 어떤 컬럼 → 엑셀 어떤 컬럼)
- [ ] 기존 데이터 처리 방식 (덮어쓰기? 아래에 추가?)
- [ ] 서식/수식 유지 여부

---

### 3. 이미지 캡처 범위
- [ ] 고정행 번호 (항상 포함되는 행, 예: 1~3행은 헤더라 항상 포함)
- [ ] 캡처 컬럼 범위 (예: A열~F열)
- [ ] 새로 입력된 데이터 행 범위 자동 감지 방식
- [ ] 이미지 저장 위치 및 파일명 규칙
- [ ] 이미지 해상도/크기 요구사항

---

### 4. 네이버카페 게시
- [ ] 카페 URL
- [ ] 게시판 이름
- [ ] 제목 생성 규칙 (예: `[YYYY.MM.DD] 입금명단 N회차`)
  - 날짜 형식:
  - 회차 계산 방식:
  - 기타 고정 문구:
- [ ] 본문 구성 (이미지만? 이미지+텍스트?)
- [ ] 태그 여부

---

### 5. 실행 스케줄
- [ ] 실행 주기 (매일 / 특정 요일 / 매월 특정일)
- [ ] 실행 시간 (예: 오전 9:00)
- [ ] 실행할 PC 환경 (항상 켜져 있나? 서버? 개인 PC?)

---

### 6. 반자동화 알림 (완전 자동화 불가 시 대안)
- [ ] 카카오톡 알림 방식
  - 개인 채팅 (KakaoTalk API 토큰 필요)
  - 카카오채널 (채널 개설 필요)
  - 오픈채팅 봇
- [ ] 알림 내용에 포함할 것 (제목 텍스트 / 이미지 미리보기 / 링크 등)
- [ ] 알림 수신 후 사용자 액션 (직접 붙여넣기 / 버튼 클릭 등)
