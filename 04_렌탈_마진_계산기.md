# 모듈 4. 렌탈 상품정리 및 마진 계산기

## 개요
렌탈(대여) 상품의 재고를 관리하고,
렌탈 요금 대비 원가 회수 기간 및 수익성을 분석하는 도구.

## 핵심 문제 정의
- 렌탈은 단순 판매와 달리 감가상각, 수리비 등 장기 비용이 발생
- 얼마나 대여해야 본전인지 파악하기 어려움
- 재고 상태(대여중/반납/수리) 추적이 수동으로 복잡

---

## 주요 기능

### 4-1. 상품 등록
- 상품명, 카테고리, 모델명, 일련번호
- 구매가, 구매일
- 사진 등록

### 4-2. 비용 구조 입력
| 항목 | 설명 |
|---|---|
| 구매가 | 초기 투자 비용 |
| 예상 내용연수 | 몇 년/몇 회 사용 가능 |
| 월 유지보수비 | 정기 점검, 소모품 비용 |
| 보험료 | 월 보험료 (선택) |
| 수리비 누계 | 발생 시마다 입력 |

### 4-3. 렌탈 요금 설정
- 일 단위 요금
- 주 단위 요금 (할인율 적용)
- 월 단위 요금 (할인율 적용)
- 보증금 설정 (선택)

### 4-4. 수익성 계산 엔진
```
총 투자비용 = 구매가 + 누적 수리비 + (월 유지보수비 × 운영 개월)
총 렌탈 수익 = 렌탈 건별 수익 합산
순수익 = 총 렌탈 수익 - 총 투자비용
손익분기점 = 총 투자비용 ÷ 일 요금 → 필요 대여일 수
```
- 현재까지 대여일 기준 회수율 (%)
- 예상 완전 회수까지 남은 기간

### 4-5. 대여 기록 관리
- 대여 등록: 고객명, 연락처, 대여 시작일, 반납 예정일, 요금
- 반납 처리: 실제 반납일, 상태 점검 결과
- 연체 계산: 반납 예정일 초과 시 자동 표시
- 대여 이력 전체 조회

### 4-6. 재고 현황 대시보드
- 전체 상품 수
- 상태별 현황: 대여가능 / 대여중 / 수리중 / 폐기
- 오늘 반납 예정 건 알림
- 연체 건 목록 강조 표시

### 4-7. 수익 분석
- 상품별 누적 수익 / 회수율 비교
- 월별 렌탈 수익 추이
- 가장 수익성 높은 상품 TOP 5
- 회수율 낮은 상품 (투자 대비 저조) 경고

---

## 화면 구성
```
[상단] 재고 현황 카드 (가능/대여중/수리중/연체)
[중간 좌] 상품 목록 테이블 (상태, 회수율 표시)
[중간 우] 선택 상품 상세 (비용구조 + 수익성 계산)
[하단] 대여 기록 + 수익 추이 차트
```
